[x] PRD-41: MILLION AGENT ARCHITECTURE. Refactor simulation to support massive scale. Create TWO agent tiers: (1) LLM-powered agents (top 30-50 key personas using gpt-4o-mini with full personality and memory), (2) Rule-based lightweight agents (thousands to millions using simple sentiment rules, no LLM calls, fast). Each lightweight agent has: archetype, stance float, influence float, response_probability float, stance_drift_rate. Each round, lightweight agents update stance based on: weighted average of nearby agent stances + random noise + archetype bias. LLM agents post real text. Lightweight agents generate simple stance votes. Aggregate both for market signal. This lets us run 1M agents with only 50 LLM calls per round.
