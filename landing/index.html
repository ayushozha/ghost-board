How Ghost Board Works

Five steps. Two feedback loops. Full autonomy. No human in the loop.

Input Concept

"Stablecoin payout API for gig workers"

AI Boardroom

5 executives build concurrently via event bus

Market Simulation

1M+ agents stress-test the product

Pivot

CEO pivots strategy, cascades to all agents

Final Artifacts

Code, financials, GTM, compliance -- all real

Inner Loop

Async Event Bus

Five agents communicate via typed pub/sub events. When Legal publishes a BLOCKER, CEO's callback fires instantly. The pivot cascades to CTO, CFO, and CMO who rebuild their artifacts in parallel.

Pivot Engine

Real Compliance Blockers

Legal agent uses OpenAI web search to cite real CFPB, FinCEN, and SEC regulations. Every blocker includes URLs to actual legal text. CEO reasons through trade-offs and pivots strategy accordingly.

Outer Loop

MiroFish Simulation

50 LLM-powered personas (VCs, journalists, competitors) and 1,000,000 lightweight NumPy-vectorized agents stress-test the product. Sentiment signals flow back to CEO for a second round of pivots.

Quick Start 

# Clone and install
$ git clone https://github.com/ayushozha/ghost-board
$ cd ghost-board && pip install -r requirements.txt

# Run the demo
$ export OPENAI_API_KEY="sk-..."
$ python main.py --demo

# Or run cached (no API key needed)
$ python main.py --cached

# Your own concept
$ python main.py "AI tutor for K-12 math"
