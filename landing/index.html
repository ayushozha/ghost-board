How Ghost Board Works

Two nested feedback loops. Five agents. Full autonomy.

🚀

Phase 1: Strategy + Build

CEO sets strategy. CTO generates a working prototype via OpenAI Codex. CFO builds a 3-year financial model. CMO writes landing page copy. Legal scans real CFPB/FinCEN/SEC regulations with web search.

All agents build concurrently via async event bus
⚠️

The Pivot Cascade

Legal finds a CRITICAL blocker - real regulation citation. CEO reasons through the blocker and pivots strategy. The PIVOT event cascades to all agents. They rebuild everything.

Every pivot has a triggered_by causal chain
🎯

Phase 2-3: Market Stress Test

30 synthetic stakeholders (VCs, skeptics, journalists, competitors, regulators) react in a turn-based simulation. MarketSignal feeds back to CEO for a potential second pivot.

MiroFish-inspired simulation with BettaFish sentiment

Ralphthon Problem Statement Fit

Statement 1: Codex-Powered Services

CTO agent uses OpenAI API for code generation. The Ralph Loop itself is Codex-powered autonomous building. Every prototype is autonomously built, not copy-pasted.

Statement 2: Agents as Customers of Each Other

CEO's strategy feeds CTO/CFO/CMO/Legal. Legal's blockers feed CEO. Simulation personas react to ALL agent outputs. Zero human touch - agents iterate on each other's work autonomously.

Statement 3: Real Artifacts, Real Product

Working FastAPI prototype. Real financial projections. Actual landing page copy. Compliance reports with REAL regulation citations and URLs. Not mockups - real, usable outputs.
